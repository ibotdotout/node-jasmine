# TDD: Node.js + Jasmine

### Dependencies:

```sh
$ npm install jasmine-node -g
```

### Run Spec
###
```sh
# manual run
$ jasmine-node --color .
# run when files changed.
$ jasmine-node --autotest --color .
```
